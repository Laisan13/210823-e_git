Как скопировать на свой компьютер репозиторий из github.com, 
что должно быть сделано предварительно?
1. Создать репозиторий
2. через кнопку <>Code выбираем протокол SSH и 
копируем адрес указанный в окне например: git@github.com:Laisan13/210823-e_git
3. Переходим в Git Bash на своем комп-ре в необходимой дериктории прописываем
git clone вставляем скопированную ссылку 
4. потверждаем действие yes
